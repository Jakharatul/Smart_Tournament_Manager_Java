Smart Tournament Manager
📌 Project Description

The Smart Tournament Manager is a Java-based application designed to automate and simplify the management of sports tournaments, especially at the college and local levels. It replaces manual methods like spreadsheets and paper records with a structured and efficient system.

The application helps in managing teams, scheduling matches, recording results, and generating leaderboards, ensuring accurate data handling and improved efficiency.

🚀 Features
✅ Team Registration and Management
✅ Player Details Handling
✅ Automatic Match Scheduling
✅ Result Recording
✅ Leaderboard Generation
✅ File Handling for Data Storage
✅ Exception Handling for Error Management
✅ Menu-driven Console Interface
🛠️ Technologies Used
Java (Core Java)
Object-Oriented Programming (OOP)
Java Collections (ArrayList)
File Handling

▶️ How to Run
Install Java (JDK 8 or above)
Open the project in any IDE (Eclipse / IntelliJ / VS Code)
Compile all .java files
Run Main.java
